# EcomTrendAI

**AIを活用したEコマーストレンド分析ツール**

Amazon、eBay、楽天市場などのEコマースプラットフォームから商品トレンドを自動収集・分析し、売れ筋商品や市場動向をリアルタイムで可視化します。

## 特徴

- **リアルタイムトレンド検出**: 急上昇商品を自動検出
- **カテゴリ別分析**: 各カテゴリの動向を把握
- **AI予測**: 将来の需要トレンドを予測（開発中）
- **自動レポート**: 日次トレンドレポートを自動生成

## クイックスタート

### 1. インストール

```bash
cd O:\Dev\Work\EcomTrendAI
pip install -r requirements.txt
```

### 2. 環境変数設定

```bash
cp .env.example .env
# .envファイルを編集してAPIキーを設定
```

### 3. 実行

```bash
python src/scraper.py      # データ収集
python src/analyzer.py     # トレンド分析
```
